create-react-app
리액트 프로젝트를 생성하는 명령줄 도구다.
개발자들이 직접 웹팩, 바벨, ESLint등 여러 도구의 설정을 손보지 않아도 빠르게 리액트 프로젝트를 시작할 수 있게 해준다.
src/App.js 파일에서 루트 컴포넌트를 수정하거나 다른 컴포넌트 파일을 임포트할 수 있다
프로젝트 폴더에서 npm start 통해 개발용 웹서버를 실행 할 수 있다. 기본 포트는 3000포트에서 실행된다
